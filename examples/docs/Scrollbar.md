## 网页自定义滚动条


### 基础用法

基础的按钮用法。
:::demo

```html
<el-row>
  <infinite-scrollbar>滚动条</infinite-scrollbar>
</el-row>
<script>
  export default {
    name: "scrollbar"
  };
</script>
```



:::
